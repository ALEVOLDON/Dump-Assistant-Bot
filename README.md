**Умный ИИ-ассистент для вашего Telegram-канала и группы обсуждений.**  
Бот работает автономно: читает ваши посты, анализирует ссылки, вовлекает аудиторию и снимает с вас рутину по ответам на частые вопросы.

---

## 🔥 Главные фишки

- 🧠 **Локальная нейросеть:** Поддержка Ollama (например, `qwen2.5:1.5b` или `7b`). Работает приватно и бесплатно. Есть fallback на OpenAI.
- 📖 **Понимание контекста:** Автоматически кэширует новые посты из канала. Когда подписчики комментируют пост, бот отвечает, опираясь на его содержимое.
- 🔗 **Чтение ссылок:** Если в посте или вопросе есть URL, бот переходит по нему, читает статью и использует эти данные для ответа.
- 🚀 **Режим «Заводилы»:** Как только публикуется новый пост, бот мгновенно оставляет первый комментарий (выжимку, вопрос аудитории или шутку), чтобы разогнать дискуссию.
- 👑 **Ко-пилот Владельца:** Бот знает вас по ID. Если пишете вы, он превращается в послушного ассистента — соглашается с вами и выполняет прямые команды.
- 🛡️ **Анти-спам и фокус:** Отвечает только на вопросы или прямые обращения. Игнорирует флуд и короткие реакции. Защищен привязкой к конкретным чатам (`ALLOWED_CHAT_IDS`).

## ⚙️ Установка и запуск

### 1. Подготовка
Убедитесь, что у вас установлен **Node.js 18+** и **Ollama** (если планируете использовать локальную модель).

```bash
# Склонируйте репозиторий и установите зависимости
npm install
```

### 2. Настройка модели
Скачайте легковесную модель для Ollama (рекомендуется `qwen2.5:1.5b-instruct` или `7b`):
```bash
ollama pull qwen2.5:1.5b-instruct
```

### 3. Конфигурация (.env)
Скопируйте пример конфига и заполните свои данные:
```bash
cp .env.example .env
```

**Обязательные переменные в `.env`:**
- `TELEGRAM_BOT_TOKEN` — токен от [@BotFather](https://t.me/BotFather)
- `BOT_USERNAME` — юзернейм вашего бота (без `@`)
- `ALLOWED_CHAT_IDS` — ID вашей группы обсуждений (узнать можно командой `/chatid`)
- `OWNER_USER_IDS` — ваш личный Telegram ID (чтобы бот вас слушался)

<details>
<summary>Пример настроек для Ollama</summary>

```env
LLM_PROVIDER=ollama
OLLAMA_MODEL=qwen2.5:1.5b-instruct
OLLAMA_BASE_URL=http://127.0.0.1:11434
OLLAMA_NUM_PREDICT=300
LLM_TIMEOUT_MS=120000
```
</details>

### 4. Подключение к Telegram
1. Зайдите в `@BotFather` -> Выберите бота -> `Bot Settings` -> `Group Privacy` -> **Turn OFF**.
2. Добавьте бота в группу обсуждений вашего канала и выдайте ему права администратора.
3. Запустите бота:

```bash
npm start
```

## 🛠 Команды управления

Пишите эти команды прямо в группу (на них ответит только бот, и только если вы Владелец):

- `/chatid` — узнать ID текущего чата (нужно для настройки `.env`)
- `/status` — показать текущий режим работы бота и статистику токенов
- `/on` — включить автоответы
- `/off` — выключить автоответы
- `/usage` — детальная статистика использования LLM

## 📝 Настройка поведения (Промпт)
Если вы хотите изменить характер бота (сделать его более дерзким, или наоборот, сухим и деловым), просто отредактируйте файл `prompts/assistant.md`. Бот подтягивает правила общения прямо оттуда.
