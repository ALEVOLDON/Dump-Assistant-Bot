Technical FAQ

What are the minimum hardware requirements? expand_more
For small models like Qwen2.5-1.5B or Llama3-3B, you need at least 8GB of RAM. A dedicated GPU (NVIDIA) with 4GB+ VRAM is recommended for instant responses, but CPU-only execution is supported through Ollama.
Can I switch from local Ollama to OpenAI? expand_more
Yes! The architecture is modular. You can update the provider settings in the `.env` file to use OpenAI or Anthropic API endpoints if you prefer cloud-based processing over local hardware.
How does it handle long articles? expand_more
The bot uses a recursive summarization technique. It fetches the link content, breaks it into manageable chunks, and creates a concise summary that fits within Telegram's message limits while preserving context.
